# Elia Evmi OCN APP

## Description
This a simple app to demonstrate OCN APP

### Installation
#### Requirements
- Node.js 14 (LTS) or higher is required.
- NPM 8.3.2 or higher
#### Run
```
npm install
npm run init
```
